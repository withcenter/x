# GnuBoard5 Extended #
-----

작성자 : 송재호

저작권 : GPL 오픈소스. 누구나 자유롭게 제한없이 이용할 수 있습니다.

이메일 : thruthesky@gmail.com

본 소프트웨어 익스텐디드(이하 'X' 로 표현)는 그누보드5 를 바탕으로 수정(또는 개조)하여 특정한 목적에 맞게 작성 한 것입니다.

해외 개발자 및 해외 사이트를 대상으로 만들다 보니 부분적으로 영어로 되어져 있습니다.
